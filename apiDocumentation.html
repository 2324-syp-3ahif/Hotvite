API Documentation

/api/user/ Endpoints

POST /signup

Creates a new user. No token required.

    Request:
    {
      "username": "testname",
      "email": "test12@gmail.com",
      "password": "123pw",
      "aboutme": "about me blabla"
    }
    
    good Responses:
    code: 201
    {
      "result": "user created"
    }

POST /login

Logs in a user. No token required.

    Request:
    {
      "email": "test12@gmail.com",
      "password": "123pw"
    }
    
    good Response:
    code: 200
    {
      "token": "generated_token",
      "username": "test name"
    }

PUT /changeUsername/

Changes the username of a user. Requires Bearer token.

    Request:
    {
      "username": "new user name"
    }
    
    good Response:
    code: 200
    {
      "result": "username changed"
    }

DELETE /delete

Deletes a user. Requires Bearer token.

    good Response:
    code: 200
    {
      "result": "user deleted"
    }
